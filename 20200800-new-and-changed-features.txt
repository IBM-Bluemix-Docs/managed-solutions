# 2020-08-00 New and changed features
{: #20200800-new-and-changed-features } 

2020-08-00 New and changed features.
{: shortdesc} 

This section describes new and changed functionality for 2020-08-00.

## Managed Applications Portal enhancements
- A link to the How-to documentation has been added to the Documentation widget in the Overview page
- The Custom views feature is now available in the Insights Portal, allowing the users to re-apply previously saved filters, sorting and pagination configurations
- The Support Portal has been enhanced with the following features:
  - Users will now be able to see the closed date of Change Requests, Service Requests and Incidents, both in the case list view and in the details page.
  - A new Copy feature was implemented to allow the creation of a new case with the information of an existing case. Users can find this option by clicking on a newly added overflow menu at the end of each row in the list view, or in the case details page. (Attachments and watchlist are not copied in this release)
  - The header of the case details page has been re-arranged to group related information together
