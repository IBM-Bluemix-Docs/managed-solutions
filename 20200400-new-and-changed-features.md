# 2020-04-00 New and changed features
{: #20200400-new-and-changed-features } 

2020-04-00 New and changed features.
{: shortdesc} 

This section describes new and changed functionality for 2020-04-00.

## Managed Applications Portal enhancements

- Following changes are made to the Alerts Widget in the Unified Portal Overview page:
  - The order in which alerts are displayed in the Alerts widget on the landing page is changed as Minor, Major and Critical
  - Alerts widget header is renamed as Open Alerts instead of Active Alerts 
  - Date picker is added to the Alerts widget to aid users select a date range and view the alerts.

- Users can see the host names (alias names) created by customers in Service Now in the Metrics Details page. 

- Free text search is added to the Add Approvers pop-up window to enable users filter down the list of available users. 

- A feature to retain page data, sort order, page size and page number, global and batch filter values if applied while 
navigating between List and Details pages on clicking the browser back button or breadcrumbs is implemented in Applications, 
Support, Servers, Insights and Health modules.  

- Following new metric types are added to the Metrics list view page:
  - Network interface input byte count (bytes)
  -	Network interface input error count (count)
  - Network interface input packet count (count)
  -	Network interface output byte count (bytes)
  - Network interface output error count (count)
  - Network interface output count (count).
